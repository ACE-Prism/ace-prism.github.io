【人物传】请叫我某J

2021-03-02 memory

  最开始是通过龙魂和墨者的视频认识某J的(我没有在迫害螺丝太太)，14年，那个时候还是在优酷看的，个人魅力实在是太强了，看播放量就知道，ra3这样小众圈子的解说视频播放量能破万已经非常牛了，但某J是至少五位数起步，有几期甚至冲到了六位数——这可是六年前啊，他就是这样一个能让完全没接触过的路人看了都乖乖交出三连的神奇男人。太牛了！

  某J影响了我很多。当时想做解说，一半是因为龙魂墨喵螺丝还有自己好奇，一半就是因为某J。如果他想，是真的可以做到破圈的——如果他继续更新的话。可能也是因为学业和环境问题，在一六年回归发了一个视频并立下flag之后，五年过去了，没有他的分毫音讯。老视频下面的评论每天都是催更，五年了几乎没怎么断过，今天凌晨刚刚上线的新视频到现在也有了一万多的播放量——评论区真就五级起步了。
  看完以后开心，激动，复杂，也很感动，还是原来的配方，还是熟悉的味道，只是我们都长大了。五年之后的今天，某J上传了RA3解说系列的最后一个视频，为自己辉煌的解说生涯划下了一个句号，也给观众朋友们，给自己的青春做了一个交代。身体的不适加上时间的匆忙，视频也许不尽完美，但是看得真的很舒畅，依然是我心中的No.1。谢谢某J，虽然看不到指挥官们给你的感谢和祝福了，但是我们2023不见不散！当然，如果不再归来，我们也会放心，这说明你在那边过得很好。愿你在自己热爱的事业中发光发热，实现理想。我们会一直记得，有这样一位指挥官，惊艳了一个时代。
  PS：以后没法催更某J了，也算是毕业了吧。这几年，接力棒逐渐交到了新人手中，我们也看到了很多优质的对战视频和mod产出。不过当年的GT社区关停确实直接影响了我的replay来源，所以后来慢慢也咕了，在nss一级组打完冠军(也是唯一一次在游戏比赛里拿到50奖金)，拿了A2+E3分级后也算基本退坑了，现在估计手生的键位都按不过来了。如果暑假有机会，可以考虑重操旧业。只是没法再现曾经的时光啦~
